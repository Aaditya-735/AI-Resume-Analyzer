# AI Resume Analyzer & Interview Coach

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using ATS scoring, skill matching, and Google Gemini.

## Features

* Resume Parsing (PDF)
* ATS Score Calculation
* Skill Extraction
* Skill Gap Analysis
* AI Resume Review using Gemini
* AI Interview Question Generator
* AI Interview Answer Evaluation
* Professional Feedback Report

## Tech Stack

* Python
* Sentence Transformers
* Google Gemini API
* PyPDF2
* Scikit-learn
* Git & GitHub
